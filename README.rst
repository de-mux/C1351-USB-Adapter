Attempt to use LUFA USB library with PlatformIO. No Arduino libraries.

Commands
========

PlatformIO must be installed::

    pip install platformio

Compile & Upload
----------------

::
    make upload

Compile only
------------

::
    make

Compile debug
-------------

::
    make debug

Generate `compile_commands.json`
--------------------------------

::
    make compiledb

Clean
-----

::
    make clean
